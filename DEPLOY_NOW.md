Your PabrikaPH project is ready to deploy in seconds!

## Click to Deploy Now 👇

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/icescream08/PabrikaPH&project-name=pabrika-ph&repo-name=PabrikaPH&env=NEXT_PUBLIC_SUPABASE_URL,NEXT_PUBLIC_SUPABASE_ANON_KEY&envDescription=Supabase%20configuration&envLink=https://supabase.com)

## What This Does

1. Clones the project from GitHub to Vercel
2. Automatically installs dependencies
3. Builds the project
4. Deploys to a live URL
5. Provides you with a public link to test

**Time to live: 2-3 minutes**

## Manual Deploy Alternative

If the button doesn't work:

1. Go to https://vercel.com/new
2. Connect your GitHub account
3. Import the PabrikaPH repository
4. Click "Deploy"
5. Add environment variables:
   - `NEXT_PUBLIC_SUPABASE_URL` (optional for testing)
   - `NEXT_PUBLIC_SUPABASE_ANON_KEY` (optional for testing)

## After Deployment

You'll get a URL like: `https://pabrika-ph.vercel.app`

Visit it to see your live PabrikaPH application!
