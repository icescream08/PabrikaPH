## 🚀 DEPLOY IN 3 MINUTES

### Option 1: One-Click Deploy (Easiest) ⭐

**Push to GitHub first (if not already done):**

```bash
# Copy project
cp -r /tmp/pabrika-ph ~/PabrikaPH
cd ~/PabrikaPH

# Push to GitHub
git remote add origin https://github.com/icescream08/PabrikaPH.git
git branch -M main
git push -u origin main
```

**Then Deploy to Vercel:**

1. Go to: https://vercel.com/new
2. Click "Import Git Repository"
3. Enter: `https://github.com/icescream08/PabrikaPH`
4. Click "Import"
5. Click "Deploy"

**That's it!** You'll have a live URL in 2-3 minutes!

### Option 2: Vercel CLI Deploy

```bash
npm install -g vercel
vercel --prod
```

Follow the prompts and your app will be live!

### Option 3: Manual Setup

1. Go to https://vercel.com
2. Sign in with GitHub
3. Click "Add New Project"
4. Import your PabrikaPH repository
5. Click "Deploy"
