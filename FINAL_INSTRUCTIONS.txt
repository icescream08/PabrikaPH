STEP 1: COPY PROJECT TO YOUR COMPUTER

ON WINDOWS (CMD):
  xcopy /tmp/pabrika-ph %USERPROFILE%\PabrikaPH /E /I
  cd %USERPROFILE%\PabrikaPH

================================================================================
STEP 2: PUSH TO GITHUB
================================================================================

EASIEST METHOD (GitHub CLI):
  1. Install GitHub CLI from: https://cli.github.com
  2. Run: gh auth login
  3. Run: gh repo create --source=. --remote=origin --push --public

OR USING HTTPS (with Personal Access Token):
  1. Create token at: https://github.com/settings/tokens
     - Name it "PabrikaPH"
     - Select scope: repo, workflow
     - Copy the token
  2. Run these commands:
     git remote add origin https://YOUR_TOKEN@github.com/icescream08/PabrikaPH.git
     git branch -M main
     git push -u origin main

OR USING SSH (if you have SSH keys set up):
  git remote add origin git@github.com:icescream08/PabrikaPH.git
  git branch -M main
  git push -u origin main

================================================================================
STEP 3: VERIFY ON GITHUB
================================================================================

Visit: https://github.com/icescream08/PabrikaPH

You should see all files and 4 commits.

================================================================================
STEP 4: SET UP LOCALLY
================================================================================

npm install
npm run dev

Visit: http://localhost:3000

================================================================================
STEP 5: DEPLOY TO VERCEL
================================================================================

1. Go to: https://vercel.com
2. Sign in with GitHub
3. Click "Add New" → "Project"
4. Search for and import "PabrikaPH"
5. Add environment variables:
   - NEXT_PUBLIC_SUPABASE_URL
   - NEXT_PUBLIC_SUPABASE_ANON_KEY
6. Click "Deploy"

You'll get a live URL automatically!
